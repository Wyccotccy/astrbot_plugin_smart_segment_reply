# 智能分段回复插件

## 插件介绍
AstrBot插件，通过调用硅基流动平台的大模型对Bot回复进行智能分段，每段随机延迟1-3秒发送，模拟真人打字节奏，提升对话自然感。

### 核心功能
- 支持6个预设免费模型，可直接复制填写使用
- 支持自定义模型名称（需硅基流动平台支持）
- 优化提示词+文本去重，彻底解决重复输出问题
